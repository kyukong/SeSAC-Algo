## ✏️ 진행방식

### 1. 일주일에 7문제 풀이하여 `PR` 제출

- 금요일까지 본인 브랜치에 코드 push 후 PR 업로드
- 다른 팀원이 확인 후 merge 진행

### 2. `issue` 를 통한 문제 토론

- 팀원들에게 공유하고 싶은 내용은 issue 에 자유롭게 작성
  - 공유하고 싶은 문제 풀이, 미해결 문제, 문제를 풀면서 얻은 인사이트 등
